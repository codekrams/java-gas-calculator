# java
-gas-calculator
Schreiben Sie ein Programm, das nach Eingabe der gefahrenen Kilometer und der Menge des verbrauchten Kraftstoffes den Durchschnittsverbrauch auf 100 km berechnet und ausgibt.

Schreiben Sie ein Programm, das nach Eingabe der gefahrenen KM und der menge des verbrauchten Sprits den Durchschnittsverbrauch auf 100 km berechnet und ausgibt. Die Berechnung darf aber nur erfolgen, wenn die gefahrenen km  größer 0 sind, ansonsten soll eine Fehlermeldung ausgegeben werden.
